# 중구 · 부산광역시

> 중구 지역 정보 랜딩 사이트

## 🌐 랜딩 페이지

### **https://kr-busan-jung.github.io/**

부산광역시 중구는 부산항을 마주 보는 원도심으로, 개항 이래 부산의 상업과 관광이 응집된 심장부 역할을 이어 온 곳입니다. 자갈치시장과 국제시장, BIFF광장, 광복로가 좁은 골목 안에 밀집해 관광객과 상인의 발길이 하루 종일 이어집니다. 면적은 약 2.82㎢로 부산의 기초자치단체 가운데 가장 작지만, 그 안에 담긴 근현대사와 시장 문화의 밀도는 어느 지역보다 짙습니다. 용두산공원 정상의 부산타워에서 내려다보면 남포동 극장가와 항만, 영도대교가 한눈에 펼쳐집니다. 6·25 전쟁기 피난 수도의 애환이 40계단과 보수동 책방골목에 고스란히 남아, 옛것과 오늘이 공존하는 독특한 정취를 만들어 냅니다.
